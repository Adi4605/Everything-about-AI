# 📘 AI Notes (Work in Progress)

This repository contains my personal notes on **Artificial Intelligence**, which I’m building while learning.

I’m using **Obsidian** to organize everything, so the notes are connected and easier to understand as a whole rather than isolated topics.

This is still a work in progress — I’ll be adding more concepts and improving existing notes over time.

---

## 🚧 What I’ll be adding

Going forward, I’ll be covering:

* Deep Learning
* Neural Networks
* Natural Language Processing (NLP)
* Computer Vision
* Reinforcement Learning
* Agentic AI
* Multiple Algorithms
* Emerging technologies

Basically, the goal is to turn this into a **complete AI knowledge base**.

---

## 📂 Structure

The structure is not fixed yet, but currently it’s something like:

* Concepts → theory
* Algorithms → models
* Metrics → evaluation
* Projects → implementations

Since this is an Obsidian vault, a lot of value comes from how notes are linked together.

---

## 🛠️ How to use this

If you want the best experience:

1. Clone the repo
2. Open it in Obsidian
3. Use graph view / backlinks

You can read it on GitHub too, but it won’t feel as connected.
