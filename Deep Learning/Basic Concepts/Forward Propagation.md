- It is the process f passing input data through a neural network to generate an output (prediction).
- 'Forward propagation is the process in which data moves from the input layer to the output layer through hidden layers, producing a prediction.'
- Components involved are:
	- Input Layer
	- Weights
	- Bias
	- Activation Function

Working:
1. Input enters network.
2. Hidden layer computes weighted sums.
3. Activation function is applied.
4. Outputs become inputs to next layer.
5. Output layer generates final prediction.

Advantages:
- Simple
- Fast
- Essential
- Works for all neural network

Common  mistakes:
- Missing bias
- Wrong activation function
- Poor weight initialization
- Data scaling issues

Complete Workflow:
Input Data
      ↓
Multiply by Weights
      ↓
Add Bias
      ↓
Apply Activation Function
      ↓
Pass to Next Layer
      ↓
Generate Output
      ↓
Prediction
