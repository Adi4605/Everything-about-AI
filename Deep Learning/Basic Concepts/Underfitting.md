- Underfitting occurs when a machine learning model is too simple too learn the underlying patterns in the data.
- 'Underfitting happens when a model fails to capture the important relationships and patterns present in the training data.'
- [[Bias-Variance Tradeoff]] for underfitting is  'High Bias + Low Variance'

Causes:
- Small model
- Too few features
- Very strong [[Regularization]]
- Insufficient Training
- Poor feature engineering

How to reduce underfitting:
- Increase model complexity
- Train longer
- Add more features
- Reduce Regularization
- Better Feature Engineering
- Use more powerful algorithms
