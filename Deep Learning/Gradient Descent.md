Gradient descent minimizes the loss function of a machine learning model.

It iteratively updates the parameters of the model in the opposite direction of the gradient of the loss function, aiming to reach the minimum of the loss function.

Variants of Gradient descent: 
- Batch Gradient Descent 
- Stochastic Gradient Descent (SGD)  
- Mini-batch Gradient Descent
