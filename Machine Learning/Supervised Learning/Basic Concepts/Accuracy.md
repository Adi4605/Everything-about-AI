It measures how many predictions made by the model are correct out of total predictions.

**"Out of all predictions, how many did the model get right?"**

Formula Of Accuracy:
Accuracy = $\frac{TP + TN}{TP + TN + FP + FN}$
where, 
- **TP (True Positive)** → Correctly predicted positive
- **TN (True Negative)** → Correctly predicted negative
- **FP (False Positive)** → Incorrect positive prediction
- **FN (False Negative)** → Incorrect negative prediction

Accuracy = $\frac{Correct Predictions}{Total Predictions}$
