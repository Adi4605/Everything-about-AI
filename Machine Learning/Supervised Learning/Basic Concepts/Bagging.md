- Bootstrap Aggregation
- It is an Ensemble Learning Technique used to improve accuracy, stability, generalization
- 'Bagging is an ensemble learning method in which multiple models are trained on different bootstrap samples of the dataset, and their predictions are combined using voting or averaging'

Working of Bagging: 
1. Create Bootstrap Samples 
2. Train Models: Train one model on each sample
3. Make Predictions: Each model predicts independently
4. Aggregate Predictions: For Classification - Majority Voting 
					For Regression - Average
