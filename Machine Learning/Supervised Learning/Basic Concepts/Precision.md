**"Out of all the samples predicted as positive, how many are actually correct ?"**

Precision = $\frac{TP}{TP+FP}$
where,
TP(True Positive) -> Correct positive predictions
FP(False Positive) -> Incorrect positive predictions

Precision answers:
- High Precision -> Few false alarms
- Low Precision -> Many wrong positive predictions

Key Points:
- Measures correctness of positive predictions
- Important when false positives matter
- Used with recall and F1 score
