- For a given input, only a small number of neurons are active (non-zero output) while most neurons remain inactive (output = 0 or near = 0)
- 'Sparse activation occurs only a few neurons "fire" for a particular input, while the majority stay inactive.'

Advantages:
- Better feature learning
- Reduced [[Overfitting]]
- Computational efficiency
- Better interpretability

Disadvantages:
- Dead neurons
- Information Loss
- [[Hyperparameter Tuning]]
