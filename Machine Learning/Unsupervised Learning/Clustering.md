Algorithm groups data points into clusters based on their similarity.
Each cluster ideally represents a group of items with common characteristics.

Clustering involves partitioning a dataset into groups or clusters, where each cluster contains data points that are more similar to each other than to those in other clusters.

Measured based on distance metrics, such as [[Euclidean Distance]] or [[Manhattan distance]], depending on the data and objective.

Clustering Algorithms:
1. [[k-means]]
2. [[Hierarchical Clustering]]

Evaluation Techniques: 
• [[Silhouette Score]]
• [[Adjusted Rand Index (ARI)]] 
• [[Normalized Mutual Information (NMI)]]
