**Eigenvalues**
Eigenvalues are unique scalar values linked to a matrix or linear transformation.
They indicate how much an eigenvector gets stretched or compressed during the transformation.
The eigenvector's direction remains unchanged unless the eigenvalue is negative, in which case the direction is simply reversed.

**Eigenvectors**
Non-zero Vectors
When multiplied by a matrix, only stretch or shrink without changing direction.
The eigenvalue must be found first before the eigenvector.
For any square matrix A of order n × n, the eigenvector is a column matrix of size n × 1. This is known as the right eigenvector, as matrix multiplication is not commutative.
