Machine Learning is a core field of AI that focuses on enabling computers to learn patterns from data and make decisions or predictions without being explicitly programmed.
Allows computers to learn from data and improve their performance automatically through experience.

"A computer program is said to learn from experience E with respect to some task T and performance measure P, if it's performance at task T, as measured by P, improves with experience E."

***Input Data + Output -> Model (Learns Rules)***
Then 
***New Data + Model -> Prediction***

Machine Learning Workflow: 
1. **Data Collection** : Gather data from sources like databases, sensors, APIs, Websites, etc.
2. **Data Processing** : Raw Data is cleaned and prepared
	- Handle missing values
	- Remove Duplicates
	- Normalize Data
	- Feature Encoding
3. **Feature Engineering** : Selecting important features from data.
4. **Model Selection** : Choose algorithm like Regression, Classification, etc.
5. **Model Training** : Train the algorithm using training data.
6. **Model Evaluation** : Check performance of the model.
7. **Model Deployment** : Deploy the model to real applications.
