*AI that doesn't just answer... it acts, plans, and complete tasks autonomously*

It is an autonomous system that act as agents, capable of planning, reasoning and taking actions to achieve specific goals with minimal human oversight.

***Understand the goal -> break it into steps -> decide actions -> execute -> observe -> improve    -> finish task***

Core Components:
1. Goal/Objective : Agent always has a goal.
2. Perception/Input : User inputs, APIs, Databases, Environment, etc.
3. Reasoning Engine : Think step-by-step, Break problems into sub-tasks, Evaluate options
4. Planning : Task decomposition, Prioritization, Scheduling
5. Action : Run Code, Call APIs, Use tools
6. Memory : Short-term memory, Long-term memory
7. Feedback loop : Learning and Adaption

Types:
1. [[Reactive Agents]]
2. [[Deliberative Agents]]
3. [[Learning Agents]]
4. [[Multi-Agent Systems]]

Architectures in Agentic AI:
- ReAct (Reason + Act) : Think -> Act -> Observe -> Think
- Plan-and-Execute : Create full plan -> Execute step-by-step
- AutoGPT-style Agents : Self-prompting, Task generation, Autonomous execution
- Tool-using Agents : Integrate with Search engines, Code interpreters, APIs
